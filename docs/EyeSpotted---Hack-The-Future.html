EyeS potted

The project

We’re leaving on an expedition into the jungle, and we want to keep track of all the animals we’ll spot. But a part of our application has been lost... Will you set up and develop the backend in Java & Spring? You will also need to make sure it can communicate with the PostgreSQL database. The entire application must be connected with the existing Angular front-end using an API. Will you make it possible to spot all the animals on our expedition?

What to do/implement

description how to setup frontend and backend

Required technologies:

- Java 17
- Spring boot
- Docker
- GIT

How to get started

  1. Pull the Front-end project from github. 
  2. Run the Docker-compose.yml to setup the database
  3. Use the spring initializr through your IDE or through https://start.spring.io/ to setup your back end project
    Add following dependencies:
    - Spring Web
    - Spring Security
    - Spring Data JPA
    - PostgreSQL Driver
    - Flyway Migration

    Other dependencies such as Lombok or modelmapper are allowed
